# Marcos Rostan — AI Engineer Portfolio

The personal website of **Marcos Rostan**, AI Engineer / LLM Engineer (Buenos Aires, Argentina). A **case-study hub** that exhibits three production-grade open-source projects, each presented with real metrics, architecture, and decisions.

**Live site:** _(coming soon — will deploy on Vercel)_

## What's here

Three case studies, each leading with production metrics, then architecture, ADR highlights, failure modes, and links to the live demo + source:

- **[claims-auditor](https://github.com/MarcosRos002/claims-auditor)** — "Veritas", a multimodal medical-billing audit agent (flagship).
- **[agent-lens](https://github.com/MarcosRos002/agent-lens)** — evaluation + observability for LLM agents.
- **[fine-tune-lab](https://github.com/MarcosRos002/fine-tune-lab)** — LoRA/QLoRA fine-tuning + distillation.

Plus a hero, about, skills, CV downloads (ES/EN), and contact links.

## Tech stack

Next.js (App Router) · TypeScript · Tailwind CSS · MDX · deployed on Vercel.
