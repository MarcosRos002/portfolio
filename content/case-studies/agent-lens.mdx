## Problem

Agents are hard to trust without measurement. You need offline eval suites,
online tracing, and latency/cost visibility — plus regression gates so quality
doesn't silently drift between commits.

## Approach

A small eval + observability toolkit: dataset-driven offline evals
(precision/recall, task-success), online tracing via LangSmith, and
latency/cost dashboards via Prometheus + Grafana. Regression gates run in CI.
